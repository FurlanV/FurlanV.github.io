Developed websites/Tools

Apr 2020 - Actually
http://ldgh.com.br/alpha/ - (currently in development)

This is a website that I'm developing for the Human Diversity Genetics Lab. It Contain two tools (also developed by me) with the objective to automatize large pipelines, organize and present human genomic data in a user friendly way to researchers.

  • First tool is the Single Nucleotide Polymorphisms Annotation Tool: This tool queries data from 13 public genomic databases and allows the user to build his own dataset, with only relevant fields for the study, dynamically merging the returned data, facilitating a long ETL and merging process that a researcher who often does not know how to program would have to go through.
  • The second tool is called DANCE (Disease ANCEstry Network). Its a tool, created by PhD. researchers on the Human Genetic Diversity Lab. This tool shows genetic relations and mutation frequency between populations, between single nucleotide polymorphisms (SNP) and complex phenotypes in a network graph approach. SNPs are nodes that are linked by edges to central nodes, that represent phenotypes.
